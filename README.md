# Manicure Bot — Telegram бот для мастера маникюра

Бот помогает маникюрному мастеру автоматизировать запись клиентов, демонстрировать портфолио и предоставлять полезную
информацию.

## Функциональность

- **Запись на приём** — выбор свободного времени из рабочего расписания, ввод контактных данных, сохранение в БД.
- **Напоминания** — автоматическое уведомление клиента за заданное количество часов.
- **Отмена записи** — клиент может отменить свою запись через бота.
- **Портфолио** — просмотр фотографий работ по категориям (Маникюр, Педикюр, Дизайн, Наращивание).
- **Информация** — прайс-лист, советы по уходу, контакты.
- **Администрирование** — добавление/удаление фото, редактирование прайса, просмотр записей, установка расписания,
  рассылка сообщений.
- **Пересылка сообщений** — все сообщения от клиентов (кроме команд) пересылаются администратору.

## Технологии

- Python 3.12
- aiogram 3.x
- SQLite + SQLAlchemy (асинхронная)
- APScheduler
- python-dotenv
